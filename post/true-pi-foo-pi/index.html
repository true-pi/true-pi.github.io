True Pi vs. Foo Pi

August 18, 2019 - 4 minutes read - 841 words

For some years now, I have been hearing that Pi ( π ) value in our textbooks is neither right nor correct.

There has been some anecdotal evidence from people manufacturing cylinders of larger diameter, saying that measured circumference is not the same as calculated one.

How did this error of Pi happen, how did it persist for over 2000 years, and still is in our textbooks today ?

But first things first, let us start with the basics.

What is Pi ?

Simply, circumference (perimeter) of a circle divided by its diameter.

Pi = π = C / d

Why should I care about the Pi ?

Pi value is everywhere.

It is in nature, it is in your calculators, computers, it is used extensively in trigonometry, geometry, statistics, signal analysis, physics, and all areas that deal with cyclical / periodic movements and some areas that do not.

And in a universe in which there are no straight lines, all is spiraling, circular motion … for any purpose in describing and modeling that universe, you would need correct value for Pi.

Currently, science is using incorrect value for Pi, let us call this incorrect value for Pi - Foo Pi.

What is the True value of Pi ?

True Pi is closely connected do another ubiquitous number found in nature - number Phi ( φ ) or Golden Ratio.

Phi is is so important that it deserves a separate article, but for now, let us just provide the value:

Phi = φ = ( 1 + √5 ) / 2 = 1,6180339887

So,

True Pi = 4 / √φ

Allow us to compare True Pi to Foo Pi (rounded to 10 decimal places):

NameValueCalculation
True Pi3,1446055110= 4 / sqrt Phi
Foo Pi3,1415926536infinite series

This means that Foo Pi is short by about 0.1% !

Difference will probably go unnoticed on low precision or small scale work, where approximate value of 3.14 is good enough.

But not in all other cases:

  • for circle with diameter d = 1 meter (approx. 3.28 feet), calculated circumference using Foo Pi will be short by 3 millimeters (approx. 1⁄8 inch) - there will be an extra 3 mm to the circumference that “science just cannot explain”.
  • if circle is size of the earth, Foo Pi calculation will be short by about 40 km (almost 25 miles).
  • and what Foo Pi does to calculations in astronomy, you can imagine for yourself. Calculations can be off by light-years, so errors can indeed become astronomical.

Unlike Foo Pi, True Pi is not a transcendental number, and squaring the circle with compass and the straight edge is easily possible.

Where did it all go so terribly wrong ?

You can find that out in detail, in our next article.

Simply, there is a fundamental error in methodology that is still intensively used in science - representing curved lines with large number of very small straight lines.

It is important to remember, so we say this a lot:

Length of a curved line cannot be correctly calculated using small segments of straight lines (rectification of a curve), no matter how small these segments are. Such method is only an approximation, and inherently contains an error.

How deep does the rabbit hole go ?

It scares to even look.

Entire infinitesimal calculus, which is significant part of mathematics and bread-and-butter of many scientists and countless engineers, does not give accurate results for this very reason.

Curved lines / surfaces cannot be correctly represented by replacing them with large number of very short straight lines / rectangles. This can give only an approximation, due to the inherent error (no matter how small the samples are).

In a universe where there are no straight lines, this may present a problem.

In Conclusion

How did this conspiracy of Pi persisted for so long (and still does) is difficult to understand.

It is unlikely that anything will change soon … due to sheer volume of things, models, software … that use this wrong Foo Pi value and many methodologies that rely on curve rectification and integral calculus.

Also some disconcerting questions arise:

  • If this simple and easily verifiable thing like Pi has eluded science for so long, what about more complex things ?
  • If we looked close enough, would we find science to have more holes than Swiss cheese ?

But I had enough for today.

Anyway, Pi is important, and you would be smart to use correct value for Pi .

For some of you that tinker in area of over-unity devices, you know that precision and coherence is everything … if you are just a little off, it does not work at all. Using correct values and understanding that comes with it, does help.

That is all.
